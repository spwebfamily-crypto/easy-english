# Prompt para Codex - Melhorar Landing Page Easy English Now

## Contexto Atual
Landing page em React + Vite para produto digital "Easy English Now" (ebook de inglês). Código está funcional com scroll reveal, glassmorphism, animações básicas. Rodando em localhost:5175.

## Objetivo
Elevar o nível visual e de conversão da landing page. Fazer parecer produto de alto valor (R$500+) não R$29. Foco em: credibilidade, desejabilidade, clareza da oferta.

- [ ] Adicionar contador de alunos/resultados ("+2.500 alunos destravaram")

- [ ] Efeito de partículas sutis no fundo (canvas/CSS)
